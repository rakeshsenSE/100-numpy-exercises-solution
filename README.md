# 100 NumPy Exercises - Solutions 🚀

This repository contains my personal solutions for the popular **"100 NumPy Exercises"** challenge. As a Computer Science and Engineering student, I am taking this challenge to master foundational Data Science, Machine Learning, and numerical computing concepts using Python's NumPy library.

## 📌 Challenge Status

- [x] Exercises 1-10: Vector creation & basic attributes
- [x] Exercises 11-20: Slicing, Reshaping & Mean values (Current)
- [ ] Exercises 21-40: Comming Soon...

## 📂 Project Structure
- `100_NumPy_exercises.ipynb`: The main Jupyter Notebook where I write, test, and run my solutions.

## 🛠️ Concepts Covered So Far
1. **Importing & Version Check:** Properly setting up NumPy and checking configurations (`np.__version__`, `np.show_config()`).
2. **Memory Size Calculation:** Understanding how arrays utilize memory in bytes (`a.size * a.itemsize`).
3. **Documentation:** Navigating NumPy functions and inspecting documentation directly from the environment using `np.info()`.

- **Language:** Python 3.x
